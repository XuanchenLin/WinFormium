项目说明

MinimalExampleApp 是一个基于 WinFormium 框架的 .NET 9 桌面应用示例项目，演示了如何在 Windows 平台下通过 HTML/CSS/JavaScript 构建现代化、可自定义的浏览器窗体界面。该项目集成了前端与后端的交互，支持多种窗口样式和系统特性，适合用作 WinFormium 的入门参考或二次开发模板。

主要功能

  • 多窗口样式演示
    • 支持通过 JavaScrispant 动态创建不同风格的演示窗口（如无边框、透明、带自定义标题栏等），并可指定窗口的背景、页面内容等参数。
  • 前后端交互
    • 通过注册 JavaScrispant 原生对象，实现前端页面与 C# 后端的同步通信，前端可调用后端方法创建新窗口等。
  • 自定义窗口行为
    • 支持窗口最大化、最小化、全屏、关闭等操作，窗口样式和行为可通过参数灵活配置。
  • 现代 UI 支持
    • 前端页面采用 HTML5、CSS3 技术，支持响应式布局和自定义主题，部分页面模拟 macOS 风格的窗口外观。
  • 资源嵌入与管理
    • 静态资源（如图片、字体、样式表等）通过 wwwroot 目录统一管理，便于扩展和维护。

适用场景

  • 需要自定义浏览器窗口的 .NET 桌面应用开发
  • 前后端混合开发、桌面与 Web 技术融合的项目原型
  • WinFormium 新手快速上手和功能演示

运行环境

以 .NET 环境运行

  • .NET 8/9
  • Windows 10/11

以 AOT 发布运行

  • Windows 7/8/8.1/10/11
